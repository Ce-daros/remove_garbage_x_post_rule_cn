# remove_garbage_x_post_rule_cn

用于「垃圾推号大扫除」的中文规则包，偏向减少误伤的升级版。

搭配脚本使用：

[垃圾推号大扫除 - Greasy Fork](https://greasyfork.org/zh-CN/scripts/573991-%E5%9E%83%E5%9C%BE%E6%8E%A8%E5%8F%B7%E5%A4%A7%E6%89%AB%E9%99%A4)

## 简介

本仓库提供一份用于屏蔽 X / Twitter 中文区垃圾黄推、引流号、博彩号、返佣号的规则文件。

相比原始规则，本版本主要调整了容易误伤正常用户的泛化正则，尤其减少对以下用户的误伤：

* 跨性别 / LGBTQ 用户名
* 日文、韩文用户名
* 使用少量 emoji、装饰符号的正常账号
* 病垢、跨圈、日圈等常见社交用户名格式

例如：

```text
清正🍥🏳️‍⚧️
⊹すずめ⊹🩵
```

## 注意

规则不保证完全准确。垃圾账号格式会变化，正常用户的命名习惯也很多样。

如果遇到误伤，欢迎提交 issue 或自行调整 `rules.json`。
